# lurkgur-react
Real-time feed of images submitted to imgur

## Requirements
* node v8.10.0+

## Optional
* yarn v1.5.1+

## Installation
```
npm install
```
or if you use yarn
```
yarn install
```

## Running
```
npm start
```
or
```
yarn start
```

## Launch
Open a browser to http://localhost:3000
